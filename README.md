## Description

Sales Order CRUD User Interface.

## Tech Stack

- VueJS
- Composition API
- Vitest
- PrimeVue
- PrimeFlex

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with Vitest

```sh
npm run test:unit
```

### Run End-to-End Tests

```sh
npm run build
npm run test:e2e
```

### Lint

```sh
npm run lint
```
